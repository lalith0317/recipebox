# RecipeBox

A full-stack recipe sharing platform built using MERN stack.

## Features
- User Authentication
- Add / Edit / Delete Recipes
- Image Upload (Cloudinary)
- Ratings & Comments
- Follow Users
- Save Recipes 

## Tech Stack
- React + Vite
- Node.js + Express
- MongoDB
- Cloudinary
